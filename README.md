# Spark Que Tech Solutions – Landing Page

A modern, fully responsive landing page built for **Spark Que Tech Solutions** as part of the internship selection round.

The website presents the company’s services, brand identity, and contact lead form in a clean, professional, and startup-style layout.

---

##  Live Preview
https://sparkquetech.netlify.app/

---

##  Tech Stack

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons  
- Vanilla JavaScript  

---

##  Features

- Fully responsive (mobile, tablet & desktop)
- Modern hero section with call-to-action buttons
- Services section with animated gradient icons
- Smooth fade-in and staggered scroll animations (no external libraries)
- Contact form with professional two-column layout
- Gradient styled buttons and social icons
- Fixed background with clean overlay
- Auto-updating footer year

---


##  Sections Included

- Hero Section  
- About Us  
- Services  
- Why Choose Us  
- Contact / Lead Form  
- Footer with social links  

All content is written meaningfully to represent a real IT services company.
